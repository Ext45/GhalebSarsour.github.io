Home Server:

In order to get more hands-on experience with networking and servers, I decided to host my own home server. I used an old 2017 MacBook Air and two 4TB external drives running Ubuntu Server (Debian had issues with MacBook Drivers). Currently, my server is running:

  • UFW
  • OpenVPN, Port-Forwarded
  • Docker Compose
  • Nextcloud
  • Minecraft Server Hosting
  • Jellyfin
